## DEMO APPLICATION

## Installation

```bash
1. git clone https://github.com/tungnm-ict/react-router-app.git
2. cd react-router-app.git
3. npm install
4. npm start
5. Open new terminal windown
npm run dev

```

## Where to look
Check `app/app.jsx` and `index.js`

Good luck!
